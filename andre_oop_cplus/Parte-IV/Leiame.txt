Na parte IV/
estão incluídos os seguintes subdiretórios:

biblioteca
	Inclue um conjunto de arquivos de código *.h *.cpp 
	e um arquivo makefile usado para compilar estes arquivos.
	O makefile possibilita a criação do executável, 
	de bibliotecas estáticas, e de biblioteca estática e dinâmica usando o libtool
	(alguns ítens do makefile executam a instalação de programas e bibliotecas, 
	para executá-los, deve-se logar como root).
	Exemplos de uso:
	make all    -> Cria executável
	make alls   -> Cria biblioteca estática e executável
	make clean  -> Limpa tudo
	make libs_libtool -> Cria biblioteca usando pacote libtool.

	O comando "make init_doc" cria arquivos de configuração para gerar a documentação, 
	e o comando "make doc" gera a documentação nos diretório html e latex
	(a documentação gerada é incompleta, pois o código ainda não foi documentado.
	O objetivo é ilustrar o uso do makefile e do doxygem).

biblioteca-Multiplataforma
	Apresenta exemplos de criação de bibliotecas multiplataforma usando ferramentas da GNU.
	normal  -> gera o executável sem usar biblioteca.
	estatico-> gera o executável usando biblioteca estática
	dinamico-> gera o executável usando biblioteca dinâmica

doc
