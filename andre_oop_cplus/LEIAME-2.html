2. INSTALAÇÃO

2.1 Download das listagens de programas:

Os arquivos com as listagens dos programas estão disponíveis para

download no link: http://www.novateceditora.com.br/downloads.php.

2.2 Para descompactar os arquivos:

Em um PC com Windows:

Use o winzip e descompacte o arquivo Livro_cpp-1.0.zip ou use o unzip:

Exemplo:

copy Livro_cpp-1.0.zip c:/tmp
cd c:/tmp
unzip Livro_cpp-1.0.zip

Em um PC com GNU/Linux:

Use o programa tar:

Exemplo:

mv Livro_cpp-1.0.tar.gz /tmp
cd /tmp
tar -xvzf Livro_cpp-1.0.tar.gz

Em uma estação de trabalho com Unix (SunOS 5.8 ou Irix64):

Exemplo:

mv Livro_cpp-1.0.tar.gz /tmp
cd /tmp
gunzip Livro_cpp-1.0.tar.gz
tar -xf Livro_cpp-1.0.tar

2.3 Para instalar o pacote de programas:

Em um PC com Windows:

Você precisa usar um compilador, a sugestão é usar o compilador da GNU disponibilizado com o ambiente de desenvolvimento Dev C++ (disponível no link http://sourceforge.net/projects/dev-cpp/).

Em um PC com GNU/Linux:

O procedimento para instalação de um programa multiplataforma distribuído com arquivos .tar.gz é o seguinte:

- Faz o download do arquivo Livro_cpp-1.0.tar.gz (no site da editora novatec).

- Descompacta o arquivo e a seguir executa a sequência ./configure && make && make install.

Exemplo:

cd /tmp/Livro_cpp-1.0
./configure
make -k
make install (como root)

Em uma estação de trabalho com Unix (SunOS 5.8 ou Irix64):

Use os programas gunzip e tar.

Exemplo:

cd /tmp/Livro_cpp-1.0
./configure
make -k
make install (como root)
